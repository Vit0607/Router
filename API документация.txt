API-ДОКУМЕНТАЦИЯ К КОМПОНЕНТУ ROUTER

Подключение:

require_once('Router.php');

При создании экземпляра класса Router в конструктор нужно передать массив из ключей-путей ('REQUEST_URI'-адресов) и значений (php-файлов, которые показывают соответствующие страницы):

Например:
$routes = [
    "/" => 'functions/homepage.php',
    "/about" => functions/about.php
];

$router = new Router($routes);

Метод компонента checkUrl() проверяет путь и подключает соответствующую страницу, либо выдает ошибку 404, если нет такой страницы.

Использование:

$router->checkUrl();
